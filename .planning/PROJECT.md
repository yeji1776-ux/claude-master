# 클로드 마스터 (Claude Master)

## 개요
AI 활용 100가지 팁을 8주 커리큘럼으로 학습하는 한국어 웹앱. 글래스모피즘 UI 테마, 모바일/데스크톱 반응형.

## 기술 스택
- **프레임워크**: Next.js 16.2.1 (App Router, React Compiler)
- **UI**: React 19.2.4 + Tailwind CSS v4
- **언어**: TypeScript ^5
- **상태관리**: React Context (타이머) + useLocalStorage 커스텀 훅
- **배포**: Vercel (자동 배포)
- **백엔드**: 없음 (클라이언트 전용, localStorage)

## 핵심 기능
1. **홈**: 대시보드 (진행률, 통계, 빠른 접근, 체크인)
2. **학습**: 8주 커리큘럼, 100개 팁, 카테고리별 진행률
3. **참고**: AG 레퍼런스, GSD 가이드, YouTube 추천
4. **도구**: 포모도로 타이머, 메모, 사전, 북마크
5. **마이**: 캘린더, 포트폴리오

## 아키텍처
- SPA 탭 기반 네비게이션 (모바일 하단바 + 데스크톱 사이드바)
- 2개 라우트: `/` (메인 쉘), `/tip/[id]` (팁 상세)
- 정적 데이터 파일 (TypeScript 배열), localStorage 영속화
- 19개 컴포넌트, 1개 Context, 1개 Hook

## 프로젝트 목표
AI 학습을 위한 올인원 웹앱으로, 학습 콘텐츠 + 생산성 도구 + 진행 추적을 통합 제공.

## 제약사항
- 백엔드/DB 없음 (클라이언트 전용)
- 외부 라이브러리 최소화 (React + Next.js + Tailwind만)
- 한국어 전용
